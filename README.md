# Web Safe Fonts

A simple web app to preview different web safe fonts, sizes and weights with jQuery.

## General Information

I have used HTML, CSS and jQuery with the DOM in mind

## Technologies

- HTML5 - DOM structure
- CSS3 - Box Positioning
- jQuery - DOM, CSS Styling with jQuery
